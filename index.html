How to code your first discord.js bot!

for linux

1. Install Visual Studio Code

- Run: sudo apt-get install code

- Run: sudo apt update

- Run: code .

After running these commands in terminal, Visual Studio Code should be updated.

2. Install node.js

- Run: sudo apt-get install nodejs

- Run: sudo apt update

Node.js should be installed now.

3. Creating projects

- Press on "file" on the top left of Visual Studio Code

- Then, press on "Open Folder"

- Name it whatever you'd like for the folder, it doesn't really matter too much.

- Press the create file button and name the file, "index.js"

- Now, create another file named: "config.json"

4. Installing dependencies

- Press on Terminal, then "New Terminal"

- Run: npm i discord.js --save

- Run: npm init

- Keep pressing enter after this until you get back to the default text.

5. Creating your bot in discord

- Open up: https://discord.com/developers

- Press "create application"

- After doing it, create a bot.

- Invite that bot by going to the OAUTH page

Here you will select "bot" and copy the invite link

After you copy it, paste it into your browser and select a test server to put it in. There! The bot has been created!

6. Putting the base commands into your bot

- Put this code into the "index.js" file:

const Discord = require('discord.js');
const { prefix, token } = require('./config.json');

const client = new Discord.Client();

client.once('ready', () => {
console.log('Ready!');
});

client.login(token);

- Then, put this file in the "config.json" file:
{
"prefix": "-",
"token": "PLACE YOUR TOKEN HERE"
}
- Make the prefix whatever you want it to be and put your bot token in the config.json file.
7. Starting the bot

Open up terminal (again) and run: node .

Go back to your server and you should see the bot online!

That's all for now!
